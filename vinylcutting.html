Vinyl cutting

The next project after football-datascience was learning the basics for the vinyl/laesr cutter
and Corell draw (the program we use to create cutting templates for the machine). The first thing we had to do was
find a raster image off the internet that we would like to cut out as a sticker (a raster image is an image entirely made out o pixels).
For my raster image, I just chose a generation 1 Decepticon logo cause I'm a Transformers nerd.

Next I copy and pasted the image into Corell vectorized the image.I then removed the image's removed the color and outlined it in black. I then took the new vector image and put it on a flash drive before taking it down to the laser/ vinyl cutter.
Next I plugged the flash drive and chose a color for the vinyl sticker while Mr. Rice set up the cutting blade on the vinyl cutter
and got the cutting template ready to go.
